lo que se necesita para ejecutar este codigo es:
minimo Python 3.11.9 
tener 10 imagenes que deben tener las siguentes nombres: carne.png, pasto.png, insectos.png, chimpance_hembra.png, chimpance_macho.png, elefante_hembra.png, elefante_macho.png, leon_hembra.png, leon_macho.png y sabana.png 
se necesita tener instalada la libreria pygame (para instalar es "pip install pygame" en la terminal)
Una vez cumplidos los pasos anteriores, abre una terminal en el directorio donde se encuentra el archivo prueba.py y ejecuta el siguiente comando: "python prueba.py"
